Fundamentos do JavaScript II – Controle de Fluxo (Condicionais e Loops)

Nesta aula, focaremos em estruturas fundamentais de controle de fluxo em JavaScript, incluindo condicionais (if/else e switch) e loops (for, while, do/while), que permitem executar blocos de código de forma seletiva ou repetitiva. Ao final, haverá exercícios práticos para consolidar o aprendizado.

1. Estruturas Condicionais (if / else)

As estruturas condicionais permitem que o programa execute diferentes blocos de código de acordo com condições específicas.

Exemplo simples de if
let tarefaConcluida = true;

if (tarefaConcluida) {
  console.log("A tarefa está concluída!");
}
Exemplo if / else
let prioridade = 2; // 1: baixa, 2: média, 3: alta

if (prioridade === 3) {
  console.log("Prioridade Alta");
} else {
  console.log("Prioridade não é alta");
}
Exemplo if / else if / else
if (prioridade === 1) {
  console.log("Prioridade Baixa");
} else if (prioridade === 2) {
  console.log("Prioridade Média");
} else if (prioridade === 3) {
  console.log("Prioridade Alta");
} else {
  console.log("Prioridade desconhecida");
}

Também é possível aninhar condicionais dentro de outras, embora seja recomendável manter o código claro para facilitar a manutenção.

2. Switch Case

A estrutura switch é útil quando há muitas comparações sobre uma mesma variável.

Exemplo de switch
let diaSemana = new Date().getDay(); // 0: Domingo, 1: Segunda, etc.

switch (diaSemana) {
  case 0:
    console.log("Domingo");
    break;
  case 1:
    console.log("Segunda-feira");
    break;
  // ...
  default:
    console.log("Dia inválido");
    break;
}

O uso de break é essencial para evitar que a execução "pule" para o próximo caso. Em alguns cenários, porém, aproveita-se a ausência de break para agrupar comportamentos semelhantes (conhecido como fall-through).

3. Loops (for, while, do/while)

Loops (estruturas de repetição) permitem executar um bloco de código múltiplas vezes, de acordo com uma condição ou até esgotar um conjunto de dados.

for

for (let i = 0; i < 5; i++) {
  console.log("Contagem:", i);
}

No loop for, temos três partes entre parênteses: inicialização (let i = 0), condição (i < 5) e incremento (i++).

while

let contador = 0;
while (contador < 5) {
  console.log("Contagem:", contador);
  contador++;
}

O while executa o bloco enquanto a condição (no caso, contador < 5) for verdadeira.

do/while

let numero = 5;
do {
  console.log("Número é:", numero);
  numero--;
} while (numero > 0);

Ao contrário do while, o do/while executa o bloco ao menos uma vez antes de verificar a condição.

4. Exercícios Práticos

Exercício 1: Condicional de Acesso

Crie uma variável nivelAcesso (1, 2 ou 3). Se o acesso for 3, exiba "Acesso Total"; se for 2, exiba "Acesso Parcial"; senão, exiba "Acesso Negado".

Ver solução
Exemplo de Resolução
let nivelAcesso = 2;

if (nivelAcesso === 3) {
  console.log("Acesso Total");
} else if (nivelAcesso === 2) {
  console.log("Acesso Parcial");
} else {
  console.log("Acesso Negado");
}

Exercício 2: Avaliação de Notas

Crie uma variável nota. Use o switch para avaliar:

  • Se 10, exiba "Nota Máxima".
  • Se 8 ou 9, exiba "Muito Bom".
  • Se 6 ou 7, exiba "Bom".
  • Se 5, exiba "Regular".
  • Caso contrário, exiba "Nota insuficiente".
Ver solução
Exemplo de Resolução
let nota = 8;
switch (nota) {
  case 10:
    console.log("Nota Máxima");
    break;
  case 9:
  case 8:
    console.log("Muito Bom");
    break;
  case 7:
  case 6:
    console.log("Bom");
    break;
  case 5:
    console.log("Regular");
    break;
  default:
    console.log("Nota insuficiente");
    break;
}

Exercício 3: Listando Números Pares

Use um for para exibir apenas os números pares de 1 a 10 no console.

Ver solução
Exemplo de Resolução
for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    console.log("Par:", i);
  }
}

Exercício 4: Soma com while

Usando um while, some todos os números de 1 a 5 e exiba o resultado.

Ver solução
Exemplo de Resolução
let soma = 0;
let contador = 1;
while (contador <= 5) {
  soma += contador;
  contador++;
}
console.log("Soma total:", soma); // 15

Exercício 5: do/while de Contagem Regressiva

Crie uma contagem regressiva usando do/while que inicie em 3 e vá até 1, exibindo "FIM!" ao término.

Ver solução
Exemplo de Resolução
let n = 3;
do {
  console.log(n);
  n--;
} while (n > 0);
console.log("FIM!");

Exercício 6: Calculadora Simples

Peça ao usuário dois valores (prompt() ou atribuições diretas) e um operador (+, -, *, /). Use switch para realizar a operação e exibir o resultado.

Ver solução
Exemplo de Resolução
let valor1 = 10; // fazendo a associação direta
let valor2 = 2;  // fazendo a associação direta
let operador = "*"; // fazendo a associação direta

//Ou peça para o usuário digitar os valores utilizando o prompt
//Só que nesse caso você precisa converter o retorno do prompt dos valores 1 e 2 para Number.

let valor1 = Number(prompt("Digite o primeiro valor"));
let valor2 = Number(prompt("Digite o segundo valor"));
let operador = prompt("Digite o operador (+, -, *, /)");

switch (operador) {
  case "+":
    console.log(valor1 + valor2);
    break;
  case "-":
    console.log(valor1 - valor2);
    break;
  case "*":
    console.log(valor1 * valor2);
    break;
  case "/":
    console.log(valor1 / valor2);
    break;
  default:
    console.log("Operador inválido");
}
