Operadores

Nesta aula, exploramos os diferentes tipos de operadores em JavaScript, fundamentais para realizar operações com valores e variáveis. Esses operadores serão aplicados no desenvolvimento do projeto TaskMaster.

  • Operadores Aritméticos: Para cálculos matemáticos (+, -, *, /, %, **)
  • Operadores de Incremento/Decremento: Para aumentar/diminuir valores (++, --)
  • Operadores de Atribuição: Para atribuir valores (=, +=, -=, *=, /=, %=, **=)
  • Operadores Relacionais: Para comparações (==, ===, !=, !==, >, <, >=, <=)
  • Operadores Lógicos: Para condições lógicas (&&, ||, !)
  • Operador Ternário: Para expressões condicionais concisas (? :)
  • Operadores Modernos: Coalescência nula (??), Encadeamento opcional (?.)
  • Operador Spread: Para expandir iteráveis (...)
  • Operadores Bit a Bit: Para manipulação binária (&, |, ^, ~, <<, >>)

Os operadores são elementos essenciais na programação JavaScript que permitem manipular valores e variáveis de diversas maneiras. Nesta aula, veremos como cada grupo de operadores funciona e como aplicá-los no contexto do TaskMaster.

Operadores Aritméticos

Usados para realizar operações matemáticas básicas entre valores numéricos. 


let a = 10;
let b = 3;

// Operações básicas
let soma = a + b;         // 13
let subtracao = a - b;    // 7
let multiplicacao = a * b; // 30
let divisao = a / b;      // 3.3333...
let modulo = a % b;       // 1 (resto da divisão)
let exponenciacao = a ** b; // 1000 (10³)

Operadores de Incremento e Decremento

Aumentam ou diminuem um valor em uma unidade.


let contador = 0;

// Pré-incremento: incrementa e depois retorna
let preIncremento = ++contador; // contador = 1, preIncremento = 1

// Pós-incremento: retorna e depois incrementa
contador = 0;
let posIncremento = contador++; // posIncremento = 0, contador = 1

// Pré-decremento: decrementa e depois retorna
contador = 2;
let preDecremento = --contador; // contador = 1, preDecremento = 1

// Pós-decremento: retorna e depois decrementa
contador = 2;
let posDecremento = contador--; // posDecremento = 2, contador = 1
++x Pré-incremento
x++ Pós-incremento
--x Pré-decremento
x-- Pós-decremento

Operadores de Atribuição

Atribuem valores a variáveis, combinando a atribuição com operações. 


let x = 10;

x += 5;  // x = 15
x -= 3;  // x = 12
x *= 2;  // x = 24
x /= 4;  // x = 6
x %= 4;  // x = 2

Operadores Relacionais (Comparação)

Comparam valores e retornam um valor booleano (true ou false).


let a = 10;
let b = 5;
let c = "10";

// Comparações
console.log(a > b);    // true
console.log(a < b);    // false
console.log(a >= b);   // true
console.log(a <= b);   // false

// Igualdades
console.log(a == b);   // false
console.log(a == c);   // true
console.log(a === c);  // false
console.log(a != b);   // true
console.log(a !== c);  // true

!== Diferente estrito (valor e tipo) 10 !== "10" true

Operadores Lógicos

Combinam expressões booleanas e retornam um valor booleano.


let verdadeiro = true;
let falso = false;

// AND (&&)
console.log(verdadeiro && verdadeiro); // true
console.log(verdadeiro && falso);      // false

// OR (||)
console.log(verdadeiro || falso);      // true
console.log(falso || falso);           // false

// NOT (!)
console.log(!verdadeiro); // false
console.log(!falso);      // true

Operador Ternário

Oferece uma forma concisa de escrever condicionais simples.


let idade = 18;
let status = idade >= 18 ? "Adulto" : "Menor de idade";
console.log(status); // "Adulto"

// Exemplo aninhado (não recomendado)
let faixaEtaria = idade < 12 ? "Criança" 
                : idade < 18 ? "Adolescente" 
                : idade < 60 ? "Adulto" 
                : "Idoso";
console.log(faixaEtaria);
condição ? expressão se verdadeiro : expressão se falso

O operador ternário é muito útil para atribuições condicionais simples, mas deve ser evitado em casos complexos para manter a legibilidade do código.

Operador de Coalescência Nula (??)

Retorna o operando do lado direito quando o operando do lado esquerdo é null ou undefined. 


let valor1 = null;
let valor2 = undefined;
let valor3 = 0;
let valor4 = "";
let valor5 = false;

let resultado1 = valor1 ?? "Valor padrão"; // "Valor padrão"
let resultado2 = valor2 ?? "Valor padrão"; // "Valor padrão"
let resultado3 = valor3 ?? "Valor padrão"; // 0
let resultado4 = valor4 ?? "Valor padrão"; // ""
let resultado5 = valor5 ?? "Valor padrão"; // false

Diferença entre ?? e ||

Enquanto o operador || considera valores "falsy" (0, "", false, NaN) para acionar o fallback, o operador ?? só utiliza o fallback para null e undefined.

Expressão ?? ||
null ?? "fallback" "fallback" "fallback"
undefined ?? "fallback" "fallback" "fallback"
0 ?? "fallback" 0 "fallback"
"" ?? "fallback" "" "fallback"
false ?? "fallback" false "fallback"

Operador de Encadeamento Opcional (?.)

Permite acessar propriedades de objetos aninhados sem gerar erro caso algum nível seja null ou undefined. 


let usuario = {
    nome: "João",
    endereco: { rua: "Av. Principal", numero: 123 }
};

let usuarioSemEndereco = { nome: "Maria" };

let ruaJoao = usuario?.endereco?.rua; // "Av. Principal"
let ruaMaria = usuarioSemEndereco?.endereco?.rua; // undefined

O operador ?. verifica cada nível antes de acessar a propriedade, prevenindo erros.

Operador Spread (...)

Permite expandir iteráveis (arrays, objetos, strings) em locais onde múltiplos elementos são esperados. 


// Arrays
let numeros1 = [1, 2, 3];
let numeros2 = [4, 5, 6];
let todoNumeros = [...numeros1, ...numeros2]; // [1,2,3,4,5,6]

// Clonagem superficial de array
let original = [1, 2, 3];
let copia = [...original];
copia.push(4);

// Objetos
let pessoa = { nome: "Ana", idade: 30 };
let pessoaComEndereco = { ...pessoa, endereco: "Rua das Flores, 123" };

Casos de Uso:

  • Concatenação de arrays
  • Clonagem superficial
  • Passar parâmetros
  • Mesclagem de objetos
  • Conversão de iteráveis

Operadores Bit a Bit

Realizam operações sobre os bits dos números, úteis para manipulação binária. 


let num1 = 5;  // 101
let num2 = 3;  // 011

console.log(num1 & num2);   // 1 (AND)
console.log(num1 | num2);   // 7 (OR)
console.log(num1 ^ num2);   // 6 (XOR)
console.log(~num1);         // -6 (NOT)
console.log(num1 << 1);     // 10 (left shift)
console.log(num1 >> 1);     // 2 (right shift)

Exemplos Práticos no TaskMaster

Como os operadores podem ser aplicados em situações reais do projeto.

Cálculo de Estatísticas de Tarefas


// Contando tarefas concluídas
let totalTarefas = tarefas.length;
let tarefasConcluidas = tarefas.filter(tarefa => tarefa.concluida).length;
let percentualConcluido = (tarefasConcluidas / totalTarefas) * 100;
console.log(`Total: ${totalTarefas}, Concluídas: ${tarefasConcluidas}, Percentual: ${percentualConcluido.toFixed(1)}%`);

Verificação de Tarefas Atrasadas


let hoje = new Date();
let dataLimite = new Date("2025-06-30");
let diasRestantes = Math.floor((dataLimite - hoje) / (1000 * 60 * 60 * 24));
let status = diasRestantes < 0 ? "Atrasada" 
           : diasRestantes === 0 ? "Vence hoje" 
           : `${diasRestantes} dias restantes`;
console.log(`Status: ${status}`);

Mesclando Dados de Tarefas


let tarefaOriginal = { id: 5, titulo: "Implementar autenticação", descricao: "Adicionar login e registro", prioridade: 2 };

Filtrando Tarefas por Critérios


let filtrarTarefas = (tarefas, { concluida, prioridadeMinima } = {}) => {
  return tarefas.filter(tarefa => {
    let passaConcluida = concluida === undefined || tarefa.concluida === concluida;
    let passaPrioridade = prioridadeMinima === undefined || tarefa.prioridade >= prioridadeMinima;
    return passaConcluida && passaPrioridade;
  });
};
let tarefasImportantes = filtrarTarefas(tarefas, { prioridadeMinima: 3 });
let tarefasPendentes = filtrarTarefas(tarefas, { concluida: false });

Desafios para Praticar

Coloque em prática o que você aprendeu resolvendo os seguintes desafios:

Desafio 1: Cálculo de Preço

Crie uma função para calcular o preço total de uma tarefa considerando o preço base e o número de horas necessárias. Cada hora adicional custa 10% do preço base.

Ver solução 

function calcularPrecoTotal(precoBase, horas) {
    let precoHoraAdicional = precoBase * 0.1;
    return precoBase + (horas * precoHoraAdicional);
}
console.log("Preço total:", calcularPrecoTotal(100, 5)); // 150

Desafio 2: Verificação de Atraso

Crie uma função que verifique se uma tarefa está atrasada, comparando a data de vencimento com a data atual.

Ver solução 

function verificarAtraso(dataVencimento) {
    let hoje = new Date();
    let dataLimite = new Date(dataVencimento);
    return hoje > dataLimite;
}
console.log("A tarefa está atrasada?", verificarAtraso("2023-01-01"));

Desafio 3: Incremento de Prioridade

Crie uma função para incrementar a prioridade de uma tarefa até um valor máximo de 5.

Ver solução 

function aumentarPrioridade(prioridadeAtual) {
    let novaPrioridade = ++prioridadeAtual;
    return novaPrioridade > 5 ? 5 : novaPrioridade;
}
console.log("Nova prioridade:", aumentarPrioridade(3));

Desafio 4: Status da Tarefa

Crie uma função que retorne o status de uma tarefa baseado em sua conclusão, utilizando o operador ternário.

Ver solução 

function statusTarefa(concluida) {
    return concluida ? "Concluída" : "Pendente";
}
console.log("Status:", statusTarefa(true));

Desafio 5: Tarefa Importante

Crie uma função que retorne se uma tarefa é importante com base em sua prioridade e prazo. A tarefa é importante se tiver prioridade maior que 3 OU estiver a menos de 2 dias do prazo.

Ver solução 

function tarefaImportante(prioridade, diasParaPrazo) {
    return prioridade > 3 || diasParaPrazo < 2;
}
console.log("Tarefa importante?", tarefaImportante(4, 5));
